# AutoKey Writer

一个 macOS 原生桌面小工具，用来把文本像人工打字一样逐字输入到当前激活窗口。
当前版本：`0.1.0`

## 功能

- 模拟人工逐字输入到当前激活窗口
- 可设置开始倒计时、换行停顿、每分钟输入字数
- 支持 `Esc` 紧急停止
- 提供权限状态提示与设置跳转
- 可打包为独立 `.app` 和分发 `.zip`

## 运行方式

```bash
./start.command
```

首次运行会自动编译原生 Cocoa 版本，之后会直接启动。

## 打包当前版本

```bash
./build_release.sh
```

打包完成后可在：

- `build/AutoKeyWriter.app`
- `dist/AutoKeyWriter-0.1.0-macOS.zip`

找到当前版本产物。

## 使用步骤

1. 启动程序。
2. 在文本框里粘贴或输入要发送的内容。
3. 设置开始倒计时和输入速度。
4. 点击“开始输入”。
5. 迅速切换到起点作家助手的目标输入框。

## 注意事项

- 目前只支持 macOS。
- 首次使用需要在“系统设置 -> 隐私与安全性”中授予辅助功能与输入监控权限。
- 本工具是向当前激活窗口发按键，所以开始后不要再点到别的地方。
- 某些编辑器会对成对符号、引号、括号有自己的自动补全逻辑，这类行为可能影响最终输入效果。
