# AutoKey Writer

一个 macOS 原生桌面小工具，用来把文本像人工打字一样逐字输入到当前激活窗口。

当前实现思路：

- 你先在本工具里准备内容
- 点击开始后有一个倒计时
- 在倒计时内把焦点切到起点作家助手的输入框
- 程序会按字符逐个发送按键

## 运行方式

```bash
./start.command
```

首次运行会自动编译原生 Cocoa 版本，之后会直接启动。

## 使用步骤

1. 启动程序。
2. 在文本框里粘贴或输入要发送的内容。
3. 设置开始倒计时和字符间隔。
4. 点击“开始输入”。
5. 迅速切换到起点作家助手的目标输入框。

## 注意事项

- 目前只支持 macOS。
- 首次使用需要在“系统设置 -> 隐私与安全性 -> 辅助功能”中允许启动它的终端，或允许编译后的 `AutoKeyWriter` 控制电脑。
- 本工具是向当前激活窗口发按键，所以开始后不要再点到别的地方。
- 中文输入依赖当前系统输入环境。通常直接发送 Unicode 文本可以工作，但不同应用兼容性可能不同。
